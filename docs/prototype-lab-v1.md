# Prototype Lab V1

`Prototype Lab` is a first internal step toward a Notion-style coded prototype workflow inside this stack.

## What V1 does

- Creates authenticated prototype projects
- Seeds a first version from a reusable template
- Stores version snapshots as file maps
- Tracks lightweight comments for review and iteration
- Stores preview and deployment URLs for sharing

## What V1 does not do yet

- Execute arbitrary code safely in sandboxes
- Generate or deploy previews automatically
- Sync directly with a production design system package
- Support multiplayer editing
- Provide a visual canvas editor

## Recommended next build phases

1. Add a real code runner with sandbox isolation.
2. Add Codex-powered file generation and edit actions.
3. Push preview builds to an internal preview environment.
4. Add richer review tools like screen annotations and approvals.
5. Connect prototypes to a shared design-system package.
